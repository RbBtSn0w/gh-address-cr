# CLI Contract: Version Query

## Global Flags

| Flag | Shorthand | Description | Expected Output |
|------|-----------|-------------|-----------------|
| `--version` | `-v` | Prints the program version and exits. | `gh-address-cr X.Y.Z` |

## Subcommands

### `version`

Prints the program version and exits.

**Usage:**
```bash
gh-address-cr version
```

**Output:**
```text
gh-address-cr X.Y.Z
```

**Exit Code:** 0
