## 2024-06-24 - Fast-path exact type checks for heavy recursive functions
**Learning:** In heavily recursive data transformation functions like JSON serialization (`json_ready`, `_json_ready`), checking exact types via `type(value) is T` for basic primitives (`str`, `int`, `float`, `bool`, `None`, standard `dict`, and `list`) is significantly faster than relying on `isinstance` checks or complex introspection functions like `is_dataclass` (which incurs `hasattr` overhead on every node). Python's `isinstance` has more overhead because it supports inheritance traversing.
**Action:** When working on serialization, traversal, or any heavy recursive function parsing raw data, add a fast-path at the top that handles the most common basic primitives using exact type matching `_type = type(value)`.
